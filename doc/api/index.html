marcos_platas_bar

Aplicación movil desarrollada en Flutter para gestionar los pedidos de un bar. Permite seleccionar los productos por mesa, ver resumen del pedido y guardarlos.

Tecnologias usadas

  • Flutter
  • Dart
  • Material Design
  • Arquitectura MVVM

Caracteristicas principales

  • Lista de pedidos por mesa
  • Creación de pedidos
  • Selección de productos y cantidades
  • Ver resumen del pedido
  • Calculo del total de productos y el precio
